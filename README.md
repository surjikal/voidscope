## hotkeys

| key        | action                       |
|------------|------------------------------|
| spacebar   | pause                        |
| mouse drag | translate X / Y              |
| arrow keys | speed X / Y                  |
| `W` / `S`  | zoom in / out                |
| `Q` / `E`  | add / remove angles          |
| hold shift | turbo mode for other actions |
